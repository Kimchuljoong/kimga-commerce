# 커머스 데모 프로젝트

**커머스 시스템 데모 프로젝트**로, **주문(Order), 고객(Member), 상품(Product)** 주요 도메인으로 구성된 커머스 데모 프로젝트

---

## **기술 스택**
- Kotlin
- Spring Boot
- JPA
- Redis
- Gradle (Multi-Module Project)

---

## **프로젝트 구조**
본 프로젝트는 **멀티 모듈 구조**로 설계되어, 각 도메인을 독립적으로 구성

- **order**: 주문 도메인
- **member**: 고객 도메인
- **product**: 상품 도메인
- **db**: 데이터베이스 관련 모듈

---

## **주요 기능**
1. **주문(Order/Stock)**
    - 주문 생성 및 조회
        - 재고 차감시 Redission을 통한 분산 Lock 적용
    - 재고 관리
2. **고객(Member)**
    - 고객 정보 관리
    - Jwt 기반 인증
    - Redis를 통한 Login 세션 관리

3. **상품(Product)**
    - 상품 등록 및 조회

---

## **향후 계획**
